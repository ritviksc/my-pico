# Interactive GPIO via Networking

A tiny, interactive tool to control and monitor GPIO pins in real time via browser.

## What it does

* Toggle GPIO pins **ON/OFF** instantly when user types GPIO Number

## Why it exists

Perfect for **learning**, **quick tests**, and **hardware debugging** using networking without writing boilerplate code.

## How to use

1. Connect your hardware
2. Run the script
3. Use the browser address to select pins and actions

## Requirements

* Supported board with wireless capabilities (e.g., Raspberry Pi / Pico W)
* Python/MicroPython (depending on target)
